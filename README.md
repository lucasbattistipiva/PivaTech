# 💻 PivaTech

Landing Page responsiva para uma loja de tecnologia desenvolvida com **HTML5**, **CSS3** e **JavaScript** puro.

O projeto possui um design moderno, navegação responsiva, menu mobile e um carrossel horizontal de produtos.

---

## 🚀 Funcionalidades

- ✅ Layout responsivo
- ✅ Menu de navegação
- ✅ Menu hambúrguer para dispositivos móveis
- ✅ Scroll suave entre seções
- ✅ Carrossel de produtos
- ✅ Cards de produtos
- ✅ Footer com informações de contato
- ✅ Design moderno

---

## 🛠 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
